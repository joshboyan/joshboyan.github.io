Appointment Ledger Map

Progressive web app built with MERN stack that schedules appointments and give users directions and timeframes.

Technology

The front-end uses React and bootstrap for a mobile-first UI. The app uses the Google Maps and Places APIs to give users autocomplete functionality and directions. The backend is an express API that prefroms CRUD operations on a mongoDB database.
